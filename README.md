# 🌾 Распознавание заболеваний сельскохозяйственных культур (Web-сервис)

![Статус проекта](https://img.shields.io/badge/Status-Beta-green)
![Python](https://img.shields.io/badge/Python-3.8+-blue)
![Django](https://img.shields.io/badge/Django-REST_Framework-red)
![YOLOv8](https://img.shields.io/badge/YOLO-v8s-yellow)

Web-сервис для автоматической идентификации заболеваний растений по фотографии с использованием технологий компьютерного зрения (CV). Проект разработан для помощи фермерам, садоводам и агрономам в быстром детектировании болезней и получении рекомендаций по лечению.

## 🌟 Возможности сервиса

* **Классификация заболеваний:** Распознавание бурой ржавчины, желтой ржавчины и септориоза на пшенице.
* **Детекция объектов:** Выделение зараженных областей на фотографии растения с помощью bounding boxes (ограничивающих рамок).
* **Множественная загрузка:** Поддержка загрузки и анализа сразу нескольких фотографий.
* **Справочник:** Выдача подробной информации о заболевании и рекомендаций по лечению конкретной болезни (если растение здорово, сервис также сообщит об этом).
* **Адаптивный дизайн:** Доступ к сервису с ПК и мобильных устройств.

## 🛠 Технологический стек

**Machine Learning / Computer Vision:**
* **Модель:** YOLOv8s (225 слоев, 11.16M параметров)
* **Библиотеки:** OpenCV (для аугментации данных и обработки изображений)
* **Инструменты разметки:** CVAT, Roboflow

**Back-end:**
* Фреймворк: Django & Django REST Framework
* База данных: PostgreSQL
* Обмен данными: REST API (передача фото в формате Base64)

**Front-end:**
* HTML5, CSS3, JavaScript
* Фреймворк: Bootstrap 5
* Дизайн: Figma

## 📊 Обучение модели и Датасет

Для обучения модели был собран датасет из открытых источников (Kaggle). Для повышения устойчивости модели была проведена аугментация (увеличение выборки в 12 раз) с использованием OpenCV: повороты на 90, 180, 270 градусов, размытие по Гауссу, изменение яркости.

**Метрики качества модели:**
* Септориоз: 0.837 mAP50
* Желтая ржавчина: 0.854 mAP50
* Общая точность (Все классы): **0.845 mAP50**

## 🚀 Установка и запуск (для разработчиков)

### 1. Клонирование репозитория
``` Bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Запуск приложения (Back-end)
``` Bash
cd web_app
python -m venv venv
source venv/bin/activate  # Для Windows: venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
